3. 약국 위치 검색
public static void pharmacy_location(Connection conn, Statement stmt)
: 약국의 이름과 약국의 주소를 출력해주는 함수이다.
쿼리에서 검색할 때 LIKE '%name%' 을 사용하였기 때문에 이름의 일부만 입력해도
입력한 문자열이 포함된 약국은 모두 출력된다.

4. 주문 추가 및 삭제
public static void medicine_order(Connection conn, Statement stmt, String clientID)
: 약 번호 찾기, 약 주문기록 조회, 약 주문, 약 주문 취소 4가지 기능이 있다.
- 약 번호 찾기: 찾고자 하는 약의 이름을 입력하면 해당하는 약의 번호를 출력해준다.
(찾은 약 번호는 추후에 약을 주문할 때 사용된다.)
- 약 주문기록 조회: 주문번호와 주문을 받았던 약사 ID를 통해서 이전에 주문했던 기록을 조회할 수 있다.
- 약 주문: 약번호, 수량, 주문날짜, 처방전 유무, 약사 ID, 주문 번호를 입력하여 약을 주문할 수 있다.
- 약 주문 취소: 주문번호와 주문을 받았던 약사 ID를 통해 이전에 했던 약 주문을 취소할 수 있다.

7. 개인정보 수정
public static void update_client_info(Connection conn, Statement stmt, String clientID)
: 로그인 때 사용했던 client_ID를 바탕으로 개인정보를 수정할 수 있다.
개인정보 수정 메뉴에 들어가면 이름, 성별, 생일, 주소, 전화번호를 조회할 수 있으며
수정할 수 있는 정보는 주소, 전화번호가 있다.

2. 약 주문 기록 조회
public static void show_medicine_history(Connection conn, Statement stmt)
: 약 이름 또는 약 번호로 해당 약을 주문했던 모든 고객들의 주문기록을 조회할 수 있다.
조회할 시 고객ID, 약 이름, 주문 날짜, 수량, 처방전 유무 등을 조회할 수 있다.
